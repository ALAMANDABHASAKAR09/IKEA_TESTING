# 🛒 **IKEA Testing Automation Project**  
📌 **A robust automation framework for testing IKEA’s APIs, UI, and workflows efficiently.**

## 📖 **Overview**
This project is designed to **test and validate IKEA’s online platform** using automation techniques. It ensures the **reliability, security, and performance** of various functionalities, including product browsing, cart operations and checkout processes.

## 🚀 **Key Features**
✅ **Cross-Browser Testing** – Ensures consistent behavior across different browsers.  

## ⚙️ **Tech Stack**
🔹 **Programming Language** – JavaScript /  Playwright / GitHub Actions / Jenkins /  Allure Reports
🔹 **Frameworks** – Playwright 
🔹 **CI/CD** – GitHub Actions, Jenkins  
🔹 **Reporting** – Allure, Extent Reports  

## 🛠️ **Installation & Setup**
1️⃣ Clone the repository:  
   ```sh
   git clone https://github.com/ALAMANDABHASAKAR09/IKEA_TESTING.git
   ```
2️⃣ Navigate to the project directory:  
   ```sh
   cd IKEA_TESTING
   ```
3️⃣ Install dependencies:  
   ```sh
   npm install  # For Node.js projects
   ```
4️⃣ Run tests:  
   ```sh
   npx playwright test || npm run test
   ```

## 📊 **Test Execution & Reporting**
 
- **Open HTML Reports:** `npm run report`
- **Open Allure Reports:** `npm run allureopen`
